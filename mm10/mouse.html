    <h2>My BigWig Track: Description and Methods</h2>


    <h3>Track Description</h3>

    <p>

        This track displays the coverage of aligned sequencing reads (bigWig format) for the LS-MPRA experiment on mouse tissues. 

        The signal values represent the normalized read depth at each genomic position.

    </p>


    <h3>Display Conventions and Configuration</h3>

    <p>

        The track is displayed in <b>full</b> mode by default. The y-axis ranges from 0 to 100, where higher peaks (darker color) indicate greater coverage. The color is set to dark blue for consistency with other signal tracks.

    </p>


    <h3>Methods</h3>

    <p>

        Raw sequencing data were aligned to the mm10 reference genome using the BWA aligner. 

        The resulting BAM file was converted to a bigWig file using the deepTools suite, removing duplicate reads. Normalization was performed using the RPKM method.

    </p>
